# 弥娅酒馆系统 - 快速开始

> 5 分钟上手弥娅酒馆系统

---

## 🚀 快速开始（3 步）

### 1. 启动酒馆
```
/start_tavern
```
弥娅会欢迎你来到深夜酒馆。

### 2. 对话
```
你好，弥娅！
```
开始与弥娅聊天。

### 3. 听故事
```
给我讲一个奇幻冒险的故事
```
聆听弥娅为你生成的故事。

---

## 📋 常用指令速查

| 指令 | 功能 | 示例 |
|------|------|------|
| `/start_tavern` | 启动酒馆 | `/start_tavern mood=warm` |
| `/tavern_chat` | 对话 | `/tavern_chat message="你好"` |
| `/generate_story` | 生成故事 | `/generate_story theme="恐怖"` |
| `/continue_story` | 续写故事 | `/continue_story` |
| `/set_mood` | 设置情绪 | `/set_mood mood=relaxed` |
| `/list_tavern_characters` | 列出角色 | `/list_tavern_characters` |
| `/create_tavern_character` | 创建角色 | `/create_tavern_character character_id=xxx` |

## 🎨 情绪氛围

| 情绪 | 说明 | 适合场景 |
|------|------|----------|
| `warm` | 温暖 | 日常聊天、闲聊 |
| `relaxed` | 轻松 | 放松、娱乐 |
| `serious` | 严肃 | 深入对话、思考 |
| `dark` | 暗黑 | 恐怖故事、悬疑 |
| `healing` | 治愈 | 倾诉、安慰 |

## 🎭 角色列表

### 弥娅 (miya)
- 身份：酒馆老板娘
- 性格：温柔、耐心、善于倾听
- 适合：日常聊天、情感交流

### 老杰克 (tavern_keeper)
- 身份：老酒保
- 性格：豪爽、幽默、见多识广
- 适合：轻松聊天、冒险故事

### 神秘旅人 (mysterious_traveler)
- 身份：旅人
- 性格：深沉、神秘、寡言少语
- 适合：哲学对话、悬疑故事

## 💡 使用技巧

### 对话技巧
- 自然交流，不用拘谨
- 可以表达真实情感
- 系统会记住上下文

### 故事技巧
- 指定清晰的主题
- 选择合适的风格
- 可以续写已有故事

### 氛围切换
- 疲惫时切换到 `healing`
- 想要放松时切换到 `relaxed`
- 需要思考时切换到 `serious`

## ❓ 常见问题

**Q: 酒馆模式和跑团模式有什么区别？**
A: 酒馆模式用于自由聊天和故事生成，跑团模式用于规则严谨的 TRPG 游戏。

**Q: 对话会被记录吗？**
A: 会，系统会记录最近 50 条对话，保持连贯性。

**Q: 可以自定义角色吗？**
A: 可以，使用 `/create_tavern_character` 创建自己的角色。

**Q: 可以导出故事吗？**
A: 故事会被保存到系统记忆中，可查看历史记录。
