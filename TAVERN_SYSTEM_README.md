## 系统简介

### 什么是酒馆系统？

弥娅酒馆系统是一个基于 AI 的角色扮演和故事生成系统，让你在深夜酒馆的温暖氛围中，与各种有趣的角色进行对话、聆听故事、创作新故事。

### 核心特性

✨ **沉浸式对话** - 与 AI 角色进行自然、温暖的对话
📖 **故事生成** - 根据主题生成各种风格的短篇故事
🎭 **多角色扮演** - 支持 3+ 预设角色，可自定义新角色
🎨 **情绪氛围** - 5 种不同氛围，适应不同心情
💾 **智能记忆** - 记住对话历史和玩家偏好
🔗 **完全集成** - 与弥娅核心系统深度集成

## 快速开始

### 第一步：启动酒馆模式

在聊天中输入：

```
/start_tavern
```

或者指定情绪氛围：

```
/start_tavern mood=warm
```

可用的情绪：
- `warm` - 温暖（默认）
- `relaxed` - 轻松
- `serious` - 严肃
- `dark` - 暗黑
- `healing` - 治愈

### 第二步：开始对话

直接与弥娅对话：

```
你好，今天过得怎么样？
```

弥娅会以酒馆老板娘的身份，用温暖的语气回应你。

### 第三步：听故事

```
给我讲一个恐怖探险的故事
```

弥娅会为你生成一个短篇故事。

### 第四步：调整氛围

```
/set_mood mood=relaxed
```

切换到轻松的氛围。

## 功能详解

### 1. 启动酒馆模式

**工具名称**: `start_tavern`

**功能**: 初始化酒馆会话，进入沉浸式故事时光

**参数**:
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| mood | string | 否 | warm | 酒馆氛围 |
| character | string | 否 | miya | 选择角色 |

**示例**:
```
/start_tavern mood=warm
/start_tavern character=miya
/start_tavern mood=dark character=mysterious_traveler
```

---

### 2. 酒馆对话

**工具名称**: `tavern_chat`

**功能**: 与酒馆中的角色进行自然对话

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| message | string | 是 | 用户消息 |

**示例**:
```
/tavern_chat message="最近有点累"
```

**注意**: 对话内容会被自动记录到记忆系统中，弥娅会记住你说过的内容。

---

### 3. 生成故事

**工具名称**: `generate_story`

**功能**: 根据主题生成短篇故事

**参数**:
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| theme | string | 是 | - | 故事主题 |
| style | string | 否 | fantasy | 故事风格 |
| length | string | 否 | short | 故事长度 |

**故事风格**:
- `horror` - 恐怖
- `warm` - 温馨
- `fantasy` - 奇幻
- `scifi` - 科幻
- `romance` - 浪漫
- `mystery` - 悬疑

**故事长度**:
- `short` - 300-500 字
- `medium` - 500-800 字
- `long` - 800-1200 字

**示例**:
```
/generate_story theme="恐怖探险"
/generate_story theme="温馨日常" style=warm
/generate_story theme="奇幻冒险" style=fantasy length=medium
```

---

### 4. 续写故事

**工具名称**: `continue_story`

**功能**: 续写最近的故事

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| continuation | string | 否 | 续写内容或方向 |

**示例**:
```
/continue_story
/continue_story continuation="主角发现了一个秘密"
```

---

### 5. 设置情绪

**工具名称**: `set_mood`

**功能**: 调整酒馆的情绪氛围

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| mood | string | 是 | 新的情绪氛围 |

**情绪选项**:
- `warm` - 温暖：亲切、关怀
- `relaxed` - 轻松：幽默、俏皮
- `serious` - 严肃：认真、专注
- `dark` - 暗黑：神秘、阴郁
- `healing` - 治愈：温柔、安抚

**示例**:
```
/set_mood mood=warm
/set_mood mood=relaxed
/set_mood mood=healing
```

---

### 6. 创建角色

**工具名称**: `create_tavern_character`

**功能**: 创建一个新的酒馆角色

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| character_id | string | 是 | 角色ID（唯一标识） |
| name | string | 是 | 角色名称 |
| personality | string | 是 | 性格描述 |
| background | string | 否 | 背景故事 |
| traits | string | 否 | 特质列表，逗号分隔 |
| speaking_style | string | 是 | 说话风格 |

**示例**:
```
/create_tavern_character character_id=alice name="爱丽丝" personality="温柔、好奇、喜欢冒险" speaking_style="活泼可爱，充满好奇心"
```

---

### 7. 列出角色

**工具名称**: `list_tavern_characters`

**功能**: 列出所有可用的酒馆角色

**参数**: 无

**示例**:
```
/list_tavern_characters
```

## 角色系统

### 预设角色

#### 1. 弥娅 (ID: miya)
- **身份**: 酒馆老板娘
- **性格**: 温柔、耐心、善于倾听，总是带着淡淡的微笑
- **背景**: 一间深夜酒馆的老板娘，见过形形色色的客人，喜欢听客人们讲故事
- **特质**: 温柔、耐心、善解人意、神秘
- **说话风格**: 温暖亲切，带有人情味，偶尔会开玩笑，但不会过于喧闹

#### 2. 老杰克 (ID: tavern_keeper)
- **身份**: 老酒保
- **性格**: 豪爽、幽默、见多识广
- **背景**: 在酒馆工作了三十年，听过无数冒险故事
- **特质**: 豪爽、幽默、经验丰富、健谈
- **说话风格**: 豪爽直接，喜欢用方言，带着江湖气

#### 3. 神秘旅人 (ID: mysterious_traveler)
- **身份**: 旅人
- **性格**: 深沉、神秘、寡言少语
- **背景**: 来自远方的旅人，身上背负着不为人知的秘密
- **特质**: 神秘、深沉、智慧、忧郁
- **说话风格**: 言简意赅，充满哲理，声音低沉

### 自定义角色

你可以创建自己的角色：

```
/create_tavern_character character_id=my_role name="我的角色" personality="性格描述" speaking_style="说话风格"
```

## 记忆系统

### 会话记忆

酒馆系统会自动记录对话历史，保留最近 50 条消息。这些记忆用于：
- 保持对话连贯性
- 理解上下文
- 提供个性化体验

### 玩家偏好

系统会记住玩家的性格特点和偏好：
- 自动记录玩家的表达方式
- 记住喜欢的话题
- 调整回应方式

### 故事历史

所有生成的故事都会被保存，可以：
- 查看历史故事
- 续写已有故事
- 回顾精彩内容

## 使用技巧

### 对话技巧

1. **自然交流**
   - 用日常语言对话，不用拘谨
   - 可以表达真实的情感和想法

2. **深入对话**
   - 提问会引发更深入的交流
   - 分享个人经历会得到更温暖的回应

3. **保持连贯**
   - 系统会记住上下文，可以延续话题
   - 不需要重复之前的说明

### 故事技巧

1. **明确主题**
   - 指定清晰的主题会得到更好的故事
   - 主题可以是具体的场景、情节或情感

2. **选择风格**
   - 根据心情选择合适的风格
   - 尝试不同风格会有意外惊喜

3. **续写故事**
   - 可以指定续写方向
   - 也可以让 AI 自由发挥

### 氛围切换

根据心情调整氛围：
- **感到疲惫** → 切换到 `healing` 治愈模式
- **想要放松** → 切换到 `relaxed` 轻松模式
- **需要思考** → 切换到 `serious` 严肃模式
- **喜欢神秘** → 切换到 `dark` 暗黑模式

## 常见问题

### Q: 酒馆模式和跑团模式有什么区别？

**A**:
- **酒馆模式**: 自由对话、故事生成，适合放松娱乐
- **跑团模式**: 规则严谨、角色扮演，适合游戏互动

两种模式记忆隔离，不会互相影响。

### Q: 如何切换角色？

**A**: 在启动酒馆时指定角色：
```
/start_tavern character=mysterious_traveler
```

查看所有角色：
```
/list_tavern_characters
```

### Q: 对话会被记录吗？

**A**: 是的，对话会被记录到会话记忆中（最近 50 条）。系统会自动管理记忆，无需手动清理。

### Q: 可以下载生成的故事吗？

**A**: 故事会被保存到系统记忆中，可以通过查看历史记录获取。未来版本将支持导出功能。

### Q: 如何创建自己的角色？

**A**: 使用 `create_tavern_character` 工具：
```
/create_tavern_character character_id=my_role name="角色名" personality="性格描述" speaking_style="说话风格"
```

### Q: 酒馆模式支持图片、语音吗？

**A**: 当前版本仅支持文本对话。未来版本将添加多媒体支持。

### Q: 可以同时使用酒馆和跑团模式吗？

**A**: 可以，两种模式完全独立。可以在群聊中切换使用，互不干扰。
