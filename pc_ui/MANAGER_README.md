# 弥娅 PC端统一管理面板

## 📋 简介

弥娅PC端统一管理面板是一个功能完整、界面美观的Web管理界面，允许你：

- 📊 查看系统整体状态
- 🎮 管理所有交互端（QQ、PC、Web）
- 🤖 管理和监控Agent
- 🧠 查询认知记忆系统
- 📈 查看队列统计信息
- ⚙️ 管理配置

---

## 🚀 快速开始

### 1. 启动弥娅后端服务

首先需要启动弥娅的Runtime API服务器：

```bash
cd d:/AI_MIYA_Facyory/MIYA/Miya

# 启动弥娅主程序（包含Runtime API）
python main.py
```

Runtime API默认运行在 `http://localhost:8000`

### 2. 打开管理面板

有两种方式打开管理面板：

#### 方式1：直接打开HTML文件

在浏览器中打开：
```
d:/AI_MIYA_Facyory/MIYA/Miya/pc_ui/manager.html
```

#### 方式2：使用本地服务器（推荐）

```bash
# 使用Python启动HTTP服务器
cd d:/AI_MIYA_Facyory/MIYA/Miya/pc_ui
python -m http.server 8080
```

然后在浏览器中访问：
```
http://localhost:8080/manager.html
```

## 🔧 配置说明

### API配置

在 `app.js` 中可以修改API配置：

```javascript
const API_CONFIG = {
    baseURL: 'http://localhost:8000',  // Runtime API地址
    endpoints: {
        probe: '/api/probe',
        status: '/api/status',
        // ... 其他端点
    },
    refreshInterval: 5000,  // 刷新间隔（毫秒）
};
```

### 修改API地址

如果Runtime API不在 `localhost:8000`，修改：

```javascript
const API_CONFIG = {
    baseURL: 'http://your-api-host:8000',
    // ...
};
```

## 🐛 故障排查

### 问题1：显示"离线"

**原因**: 无法连接到Runtime API

**解决方案**:
1. 检查Runtime API是否已启动
2. 检查API地址配置是否正确
3. 检查浏览器控制台是否有CORS错误

### 问题2：交互端无法启动/停止

**原因**: Runtime API端点未正确配置

**解决方案**:
1. 检查Runtime API是否实现了对应端点
2. 检查浏览器控制台错误信息
3. 检查弥娅后端日志

### 问题3：数据不更新

**原因**: 自动刷新失败

**解决方案**:
1. 手动点击刷新按钮
2. 检查网络连接
3. 检查API端点是否正常
