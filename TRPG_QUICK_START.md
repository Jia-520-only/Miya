# 弥娅 TRPG 跑团系统 - 快速开始

## 🎲 如何测试

### 方法 1: 运行测试脚本

在项目根目录运行：

```bash
python test_trpg_system.py
```

或使用批处理文件：

```bash
test_trpg.bat
```

测试脚本会自动执行：
1. ✅ 启动跑团模式
2. ✅ 创建角色卡
3. ✅ 查看角色卡
4. ✅ 投骰功能
5. ✅ 暗骰功能
6. ✅ D&D 5E 规则测试
7. ✅ COC 7 规则测试
8. ✅ 骰子系统测试

### 方法 2: 仅测试骰子系统

```bash
python test_trpg_system.py --dice-only
```

这会快速测试各种骰子表达式：
- d100, d20, 3d6, 2d10+5, 4d6-2, 1d8+3

## 🎮 在 QQ 群中使用

### 基础指令

```
/trpg coc7 团名              # 启动跑团模式
/roll 3d6                   # 投骰
/roll 3d6 力量检定           # 带原因的投骰
/roll_secret 1d100 潜行       # 暗骰（仅 KP 可见）
```

### 角色卡指令

```
/pc create 角色名 coc7       # 创建 COC7 角色卡
/pc create 角色名 dnd5e      # 创建 D&D 5E 角色卡
/pc show                    # 查看自己的角色卡
/pc show 123456              # 查看指定玩家的角色卡
```

## 📁 数据文件

测试会创建以下文件：

- `data/trpg_sessions.json` - 跑团会话数据
- `data/trpg_characters.json` - 角色卡数据

如需重置测试数据，删除这些文件即可：

```bash
rm data/trpg_sessions.json data/trpg_characters.json
```

## 🔧 故障排除

### 问题：UnicodeEncodeError

**原因**：Windows 终端编码问题

**解决**：测试脚本已自动处理，如果仍有问题，确保使用 UTF-8 编码

### 问题：工具未注册

**原因**：模块导入失败

**解决**：
```bash
# 检查工具是否注册
python -c "from webnet.ToolNet.registry import ToolRegistry; r = ToolRegistry(); r.load_all_tools(); print([k for k in r.tools.keys() if 'trpg' in k.lower()])"
```

应该看到：`['start_trpg', 'roll_dice', 'roll_secret', 'create_pc', 'show_pc']`
