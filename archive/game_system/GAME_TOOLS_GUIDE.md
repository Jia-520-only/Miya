# 游戏工具使用指南

## 📖 概述

弥娅的游戏系统包含多种工具，用于管理跑团、酒馆模式以及相关的存档和检索功能。本指南详细介绍了每个工具的使用方法。

### 2. `create_game_save` - 创建游戏存档

**功能：** 手动保存当前游戏状态

**使用方式：**
- "保存游戏"
- "创建存档"
- "保存当前进度"

**可选参数：**
- `save_name`: 存档名称（如果不提供，系统会自动生成）

**使用示例：**
- "保存游戏为【第一章：冒险开始】"
- "创建名为【我的第一次跑团】的存档"

---

### 3. `load_game_save` - 加载游戏存档

**功能：** 加载指定的游戏存档

**使用方式：**
- "加载存档ID xxxxxx"
- "读取存档 xxxxxx"
- "恢复游戏到存档 xxxxxx"

**参数：**
- `save_id`: 必需，存档ID（通过 `list_game_saves` 查看获取）

**使用示例：**
- "加载存档ID 12345678-1234-1234-1234-123456789abc"
- "读取 12345678-1234-1234-1234-123456789abc 这个存档"

## 🔍 跑团检索工具 (TRPG Query Tools)

### 4. `search_trpg_characters` - 搜索跑团角色卡

**功能：** 搜索跑团角色卡，支持按角色名、技能、装备、规则系统等条件搜索

**使用方式：**
- "查找名为亚瑟的角色卡"
- "搜索有侦查技能的角色"
- "显示所有coc7规则的角色"

**可选参数：**
- `query`: 搜索关键词（角色名、技能名、装备名等）
- `rule_system`: 规则系统（coc7、dnd5e等）
- `group_id`: 群组ID（限定特定群组的角色）
- `limit`: 返回结果数量（默认10，最大50）

**使用示例：**
```
"搜索亚瑟的角色卡"
"查找有剑的角色"
"显示所有coc7角色，返回20个"
```

### 5. `search_trpg_by_attribute` - 按属性值搜索角色卡

**功能：** 查找属性值达到特定条件的角色

**使用方式：**
- "查找力量大于80的角色"
- "显示敏捷最高的角色"
- "搜索体质90以上的角色"

**必需参数：**
- `attribute_name`: 属性名称（strength/dexterity/constitution/intelligence/wisdom/charisma）
- `min_value`: 最低属性值

**可选参数：**
- `rule_system`: 规则系统（coc7、dnd5e等）
- `group_id`: 群组ID
- `limit`: 返回结果数量（默认10）

**属性名称对照表：**
| 英文 | 中文 |
|------|------|
| strength | 力量 |
| dexterity | 敏捷 |
| constitution | 体质 |
| intelligence | 智力 |
| wisdom | 感知 |
| charisma | 魅力 |

**使用示例：**
```
"查找力量大于80的角色"
"显示体质90以上的coc7角色"
"搜索智力高于70的角色"
```

## 💡 使用技巧

### 最佳实践

1. **组合搜索：** 可以结合多个条件进行精确搜索
   ```
   "查找coc7规则下力量大于80且有剑术技能的角色"
   ```

2. **自然语言：** 可以使用自然语言描述，AI会自动理解意图
   ```
   "我想看看之前跑团时创建的角色卡"
   "帮我找找之前冒险的存档"
   ```

3. **分步查询：** 对于复杂需求，可以分步进行
   ```
   "先显示所有存档"
   "然后加载第一个存档"
   ```

### 常见问题

**Q: 如何知道我的存档ID？**
A: 使用 `list_game_saves` 命令查看所有存档，每个存档都有唯一的ID。

**Q: 存档会自动保存吗？**
A: 是的，游戏会自动保存。你也可以使用 `create_game_save` 手动创建命名存档。

**Q: 搜索结果太多怎么办？**
A: 可以使用 `limit` 参数限制返回数量，或者添加更多筛选条件。

**Q: 可以跨游戏类型搜索吗？**
A: 可以，使用 `rule_system` 参数限定特定规则系统（coc7、dnd5e、tavern等）。

## 🔧 故障排除

**问题：工具调用次数过多**
- 解决方案：已修复，将 `max_iterations` 从10增加到20

**问题：必须提供 group_id 或 user_id**
- 解决方案：已修复，工具会自动从上下文获取这些参数

**问题：无法加载存档**
- 解决方案：检查存档ID是否正确，使用 `list_game_saves` 查看可用存档

**问题：搜索结果为空**
- 解决方案：
  - 检查搜索关键词是否正确
  - 确认是否有相关数据存在
  - 尝试使用更通用的关键词
