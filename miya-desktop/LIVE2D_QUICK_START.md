# Live2D 独立窗口 - 快速启动指南

## 🚀 5 分钟快速上手

### 步骤 1: 确认库文件已下载

检查 `public/libraries/` 目录下是否有以下文件：
- ✅ `pixi.min.js` (460 KB)
- ✅ `pixi-live2d-display.min.js` (126 KB)
- ✅ `live2dcubismcore.min.js` (207 KB)

如果没有，运行下载脚本：
```bash
powershell -ExecutionPolicy Bypass -File download_libraries.ps1
```

### 步骤 2: 启动应用

**方法 A: 使用测试脚本（推荐）**
```bash
test_live2d_standalone.bat
```

**方法 B: 手动启动**
```bash
# 启动开发服务器
npm run dev

# 新开终端，启动 Electron
npm run dev:electron
```

### 步骤 3: 打开 Live2D 控制面板

1. 应用启动后，进入聊天界面
2. 点击左侧工具栏的 **机器人图标** 🤮
3. Live2D 控制面板将从右侧滑出

### 步骤 4: 创建独立窗口

在控制面板中：
1. 点击 **"打开窗口"** 按钮
2. 一个新的 Live2D 窗口将显示在桌面上
3. 窗口可以在桌面上自由拖动

### 步骤 5: 尝试功能

#### 拖动窗口
- 点击窗口任意位置并拖动
- 窗口会跟随鼠标移动

#### 调整大小
- **方法 1**: 拖动窗口边缘
- **方法 2**: 使用控制面板的滑块
  - 宽度：200px - 800px
  - 高度：300px - 900px

#### 切换表情
在控制面板的表情网格中点击：
- 😊 开心
- 😳 害羞
- 😠 生气
- 😢 悲伤
- 😐 平静
- 🤩 兴奋
- 😜 调皮
- 🤫 嘘声

#### 快捷预设
点击控制面板底部的预设按钮：
- 📱 **小窗口** - 300x400px
- 💻 **中窗口** - 400x500px
- 🖥️ **大窗口** - 600x700px
- 📺 **全屏模式** - 800x900px

#### 窗口置顶
点击 **"窗口置顶"** 按钮，切换窗口是否始终在最上层

## 🔧 常见问题

### Q1: 点击"打开窗口"没有反应？
**A**:
1. 检查控制台是否有错误信息
2. 确认库文件已下载到 `public/libraries/`
3. 尝试重启应用

### Q2: 表情不切换？
**A**:
1. 确认 Live2D 窗口已打开
2. 检查控制面板的状态信息
3. 尝试点击其他表情按钮

### Q3: 窗口无法拖动？
**A**:
1. 确认窗口是独立窗口（不是嵌入在主界面中）
2. 尝试点击窗口中央位置拖动
3. 检查 CSS 中 `-webkit-app-region: drag` 是否生效

### Q4: 窗口大小无法调整？
**A**:
1. 确认窗口是独立窗口
2. 尝试使用控制面板的滑块调整
3. 检查窗口配置中的 `resizable: true`

### Q5: 窗口被其他窗口遮挡？
**A**:
1. 点击控制面板的"窗口置顶"按钮
2. 窗口将始终显示在最上层
