# 桌宠调整大小功能说明

## 功能概述

桌宠窗口现在支持自定义调整大小，你可以将桌宠设置为任意想要的大小。

## 新增功能

### 1. 窗口调整大小
- ✅ 通过拖拽窗口边缘调整大小
- ✅ 右侧边缘：调整宽度
- ✅ 底部边缘：调整高度
- ✅ 右下角：同时调整宽度和高度

### 2. 调整大小的交互区域
窗口有三个透明的调整手柄：
- **右侧边缘**：鼠标悬停时显示光标 `↔`，拖动可调整宽度（最小 300px）
- **底部边缘**：鼠标悬停时显示光标 `↕`，拖动可调整高度（最小 400px）
- **右下角**：鼠标悬停时显示光标 `↘`，拖动可同时调整宽度和高度

### 3. 鼠标穿透优化
- **默认状态**：桌宠完全透明，鼠标穿透点击后面内容
- **调整大小时**：临时禁用鼠标穿透，允许拖拽
- **调整完成后**：自动恢复鼠标穿透

### 4. 无边框设计
- ✅ 完全无边框，只有 Live2D 形象
- ✅ 没有任何控制按钮或标题栏
- ✅ 真正的"悬浮"效果

## 使用方法

### 方法 1：拖拽边缘（推荐）
1. 将鼠标移动到窗口右边缘、底边缘或右下角
2. 鼠标会变成调整大小的光标
3. 按住并拖动到想要的大小
4. 释放鼠标即可

### 方法 2：Shift + 拖动移动窗口
1. 按住键盘上的 `Shift` 键
2. 点击并拖动 Live2D 模型
3. 将窗口移动到任意位置

### 方法 3：滚轮缩放
1. 将鼠标悬停在 Live2D 模型上
2. 滚动鼠标滚轮
3. 向上滚动放大，向下滚动缩小

## 技术实现

### 文件修改列表
1. `electron/modules/live2d-window.ts`
   - 启用 `resizable: true`
   - 添加 `getLive2DWindowSize()` 函数

2. `electron/main.ts`
   - 添加 `live2d:getSize` IPC 处理器

3. `electron/preload.ts`
   - 添加 `getSize()` 方法到 Live2D API

4. `src/electron.d.ts`
   - 添加 `getSize()` 类型声明

5. `src/views/Live2DStandalone.vue`
   - 添加调整大小手柄 UI
   - 实现拖拽调整大小逻辑
   - 优化鼠标穿透控制

## 注意事项

1. **最小尺寸限制**
   - 宽度最小：300px
   - 高度最小：400px

2. **鼠标穿透行为**
   - 调整大小时会临时禁用穿透
   - 完成后自动恢复穿透
   - 如果操作卡住，可以重新打开桌宠

3. **性能优化**
   - 调整大小时窗口大小会实时更新
   - Live2D 模型会自动适应新的窗口大小
