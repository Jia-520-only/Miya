## 📌 注意事项

### 开发规范
1. **代码规范**
   - 遵循 ESLint 规则
   - 使用 TypeScript 类型
   - 组件命名规范

2. **Git 提交规范**
   - feat: 新功能
   - fix: 修复 bug
   - style: 样式调整
   - refactor: 重构
   - test: 测试
   - docs: 文档

3. **代码审查**
   - 每个 PR 需要至少一人审查
   - 通过所有测试才能合并
   - 更新相关文档

### 性能指标
- 首屏加载 < 2s
- 交互响应 < 100ms
- 内存占用 < 500MB
- CPU 占用 < 10%

### 兼容性
- Chrome/Edge >= 90
- Firefox >= 88
- Safari >= 14
- 系统版本 >= Windows 10 / macOS 11
