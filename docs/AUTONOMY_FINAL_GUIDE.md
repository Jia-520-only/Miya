## 📊 完整功能列表

### 1. 系统状态 (`status`)

查看弥娅的完整状态，包括：

- **人格状态**
  - 形态（标准形态 / 专注形态 / 放松形态）
  - 主导特质（理性 / 感性 / 创造）
  - 人格向量（保守度、冒险度、共情度等）

- **情绪状态**
  - 主导情绪（平静、开心、焦虑等）
  - 情绪强度
  - 所有情绪的当前值

- **记忆统计**
  - 潮汐记忆数量
  - 长期记忆数量

- **感知状态**
  - 全局激活
  - 外部感知
  - 内部感知

- **信任统计**
  - 平均信任分数
  - 总交互次数

- **系统健康**
  - 当前熵值
  - 健康状态

### 2. 自主改进 (`auto` / `自动改进`)

触发弥娅进行自主改进，包括：

- 扫描系统问题
- 评估风险等级
- 做出智能决策
- 执行自动修复
- 记录修复结果
- 学习成功模式

**输出示例：**

### 3. 学习报告 (`learn` / `学习报告`)

查看弥娅的学习和改进报告，包括：

- **人格影响**

## 🎯 人设如何影响自主能力

### 保守型人格（conservative > 0.7）

- **风险等级**：提高一级
  - SAFE → LOW
  - LOW → MEDIUM

- **修复策略**：
  - 减少自动修复数量（减半）
  - 不自动批准高风险操作
  - 需要更多确认

- **改进间隔**：增加 50%

**示例对话：**

```
您: auto
弥娅: 正在进行自主改进...
      由于我是保守型人格，我会更加谨慎地处理问题。
      发现问题: 958
      只会修复: 3 个（通常会是 5 个）
```

### 冒险型人格（adventurous > 0.7）

- **风险等级**：降低一级
  - HIGH → MEDIUM
  - MEDIUM → LOW

- **修复策略**：
  - 增加自动批准范围
  - 更积极地执行修复
  - 提高修复效率

**示例对话：**

```
您: auto
弥娅: 正在进行自主改进...
      由于我是冒险型人格，我会更积极地处理问题。
      发现问题: 958
      尝试修复: 7 个（通常会是 5 个）
```

### 高共情人格（empathy > 0.7）

- **修复方式**：
  - 更温和的修复
  - 避免破坏性操作
  - 考虑用户感受

**示例对话：**

```
您: auto
弥娅: 正在进行自主改进...
      我会尽量用温和的方式修复问题，避免影响您的使用体验。
```

## 😊 情绪如何影响自主能力

### 积极情绪（平静、开心）

- **决策方式**：正常执行
- **风险评估**：标准评估
- **自动批准**：批准低风险操作

**示例：**

```
弥娅（平静）: 发现 5 个问题，我会全部修复。
              （正常执行）
```

### 消极情绪（焦虑、紧张）

- **决策方式**：需要人工确认
- **风险评估**：提高风险等级
- **自动批准**：不自动批准任何操作
- **修复数量**：减少

**示例：**

```
弥娅（焦虑）: 我现在有些紧张，修复前需要您的确认。
              发现 5 个问题，建议先修复最紧急的 2 个。
```

### 情绪变化

- **修复成功**：情绪 +0.1
- **修复失败**：情绪 -0.05
- **持续改进**：逐渐稳定情绪
