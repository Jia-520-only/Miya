# 弥娅代码编辑器快速启动指南

## 🚀 5分钟上手

### 1. 启动应用

```bash
# 在项目根目录执行
start_desktop_smart.bat
```

或手动启动：

```bash
# 终端1：启动后端
python run/main.py

# 终端2：启动前端
cd miya-desktop
npm run dev

# 终端3：启动 Electron
cd miya-desktop
npm run dev:electron
```

### 2. 打开代码编辑器

1. 弥娅桌面应用启动后，会默认显示"对话"页面
2. 点击顶部导航栏的 **"代码"** 标签
3. 即可进入代码编辑器界面

### 3. 第一个程序

1. 点击工具栏的 **"新建文件"** 按钮
2. 输入文件名：`hello.js`
3. 在编辑器中输入以下代码：

```javascript
console.log("Hello, Miya!");
console.log("欢迎来到弥娅代码编辑器！");

for (let i = 1; i <= 5; i++) {
  console.log(`计数: ${i}`);
}
```

4. 点击工具栏的 **"运行"** 按钮（绿色播放图标）
5. 在底部终端面板查看输出结果

### 4. 编辑现有文件

1. 在左侧文件树中找到您想编辑的文件
2. 点击文件名即可打开
3. 在编辑器中修改代码
4. 点击 **"保存"** 按钮（蓝色保存图标）
5. 文件名旁边的 ● 标记会消失，表示已保存

## 🎯 核心功能速查

| 功能 | 操作 | 说明 |
|------|------|------|
| **打开文件** | 点击左侧文件树 | 从文件资源管理器打开 |
| **新建文件** | 点击 [+新建] 按钮 | 创建新代码文件 |
| **保存文件** | 点击 [💾保存] 按钮 | 保存当前更改 |
| **运行代码** | 点击 [▶运行] 按钮 | 执行当前代码 |
| **格式化代码** | 点击 [📝格式化] | 自动格式化代码 |
| **切换终端** | 点击 [🖥终端] 按钮 | 显示/隐藏终端面板 |
| **筛选文件** | 在筛选框输入 | 快速查找文件 |
| **切换语言** | 点击语言下拉框 | 选择代码语言 |

## 💡 常用技巧

### 1. 快速保存
编辑文件后，文件名旁会出现 **●** 标记，表示未保存。点击"保存"按钮后标记消失。

### 2. 代码格式化
编辑完代码后，点击"格式化"按钮，代码会自动对齐和缩进。

### 3. 终端查看
运行代码后，点击"终端"按钮打开底部面板，可以查看执行结果。

### 4. 多语言支持
点击语言下拉框，支持以下语言：
- TypeScript / JavaScript
- Python
- Java
- C / C++ / C#
- Go / Rust
- PHP / Ruby
- HTML / CSS
- JSON / Markdown / SQL

### 5. 文件筛选
在文件树顶部的筛选框中输入文件名，快速查找文件。

## 🔧 故障排除

### 问题1: 点击"运行"没有输出
**解决方案**: 确保点击了"终端"按钮打开底部面板，输出会在终端或输出标签中显示。

### 问题2: 文件无法保存
**解决方案**: 确保后端服务正在运行，文件有写入权限。

### 问题3: 编辑器显示空白
**解决方案**: 尝试刷新页面或重启应用。确保 Monaco Editor 已正确安装。

### 问题4: 文件树为空
**解决方案**: 这是正常的初始状态，可以点击"新建文件"创建新文件，或从其他视图打开文件。
