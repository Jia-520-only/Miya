# 弥娅 TRPG 跑团系统 - 核心框架完成

## ✅ 已完成的功能

### 1. 骰子系统 (`webnet/EntertainmentNet/trpg/dice.py`)
- ✅ 支持标准骰子表达式：`d100`, `3d6`, `2d10+5`, `4d6-2`
- ✅ 骰子解析引擎
- ✅ 投骰结果详细显示

**测试结果：**
```
d100: 1d100=[14] = 38
3d6: 3d6=[5+6+6] = 17
2d10+5: 2d10=[8+1]+5 = 14
```

### 2. 会话管理系统 (`webnet/EntertainmentNet/trpg/session.py`)
- ✅ 三种 KP 模式支持：
  - `independent`: 每个群独立 KP
  - `cross_group`: 跨群共享 KP
  - `global`: 全局唯一 KP
- ✅ 会话持久化存储
- ✅ 玩家管理

### 3. 角色卡系统 (`webnet/EntertainmentNet/trpg/character.py`)
- ✅ 支持 COC7 和 D&D 5E 规则
- ✅ 角色属性随机生成
- ✅ 跨群角色卡共享
- ✅ 按群隔离的角色状态
- ✅ 角色卡持久化存储

### 4. 规则系统 (`webnet/EntertainmentNet/trpg/rules/`)
- ✅ COC7 规则模块：
  - d100 判定系统
  - 成功等级判定（大成功、极难成功、困难成功、成功、失败、大失败）
  - 衍生属性计算
  - 理智检定
- ✅ D&D 5E 规则模块：
  - 属性修正值计算
  - 属性检定（支持优势/劣势）
  - 技能检定
  - 攻击检定
  - 伤害检定
  - 先攻检定

### 5. 基础工具 (`webnet/EntertainmentNet/trpg/tools/`)
- ✅ `start_trpg` - 启动跑团模式
- ✅ `roll_dice` - 投骰工具
- ✅ `roll_secret` - 暗骰工具（结果发送给 KP）
- ✅ `create_pc` - 创建角色卡
- ✅ `show_pc` - 查看角色卡

## 🎮 使用示例

### 1. 启动跑团模式
```
用户: /trpg coc7 迷雾图书馆
弥娅: 🎲 跑团模式已启动
规则系统：COC 7版
团名称：迷雾图书馆

可用指令：
• /roll 3d6         - 投骰
• /sc 侦查 70        - 技能检定
• /rs 1d100 侦查     - 暗骰
• /pc create 名字    - 创建角色卡
• /pc show           - 查看角色卡
• /kp set_mode       - 设置 KP 模式
```

### 2. 创建角色卡
```
用户: /pc create 亚瑟 coc7
弥娅: ✅ 角色卡创建成功！

👤 角色：亚瑟
🎮 规则：COC7

【基础属性】
力量： 55  敏捷： 62  体质： 48
外貌： 70  智力： 65  意志： 68
幸运： 50  教育： 60
```

### 3. 投骰
```
用户: /roll 3d6 力量检定
弥娅: 🎲 力量检定
3d6=[2+4+1] = 7
```

### 4. 暗骰（仅 KP 可见结果）
```
用户: /roll_secret 1d100 侦查
群内: 🎲 用户 123456 进行暗骰：侦查

KP 私聊: 🔒 暗骰结果
        玩家：123456
        原因：侦查
        结果：1d100=[42] = 42
```
