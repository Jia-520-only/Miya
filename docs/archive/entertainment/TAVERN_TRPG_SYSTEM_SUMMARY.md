## 🎯 系统概览

弥娅娱乐系统包含两个互补的娱乐模式：

### 1. TRPG 跑团系统
- **用途**：规则严谨的桌面角色扮演游戏
- **规则**：支持 COC7、D&D5E
- **核心**：骰子、角色卡、场景管理、KP 系统

### 2. 酒馆故事系统
- **用途**：自由对话和故事生成
- **特色**：AI 角色扮演、故事创作
- **核心**：对话、故事、角色、情绪

## 🎲 TRPG 跑团系统

### 工具列表 (11 个)

| 工具 | 功能 |
|------|------|
| `start_trpg` | 启动跑团模式 |
| `roll_dice` | 公开投骰 |
| `roll_secret` | 暗骰 |
| `create_pc` | 创建角色卡 |
| `show_pc` | 查看角色卡 |
| `update_pc` | 更新角色卡 |
| `delete_pc` | 删除角色卡 |
| `skill_check` | 技能检定 |
| `kp_command` | KP 指令 |
| `attack` | 攻击检定 |
| `combat_log` | 战斗日志 |

### 核心功能

✅ **骰子系统**
- 支持标准骰子表达式：1d100、3d6、2d10+5
- COC7 规则判定
- D&D5E 优势/劣势

✅ **角色卡系统**
- 支持跨群状态隔离
- 属性、技能、装备管理
- 3 种 KP 模式

✅ **规则系统**
- COC7：d100 检定、理智系统
- D&D5E：d20 检定、修正值

✅ **场景与剧情**
- 场景管理
- NPC 管理
- 线索系统
- 战斗系统

## 🔧 技术特点

### 架构设计

✅ **子网分离**
- TRPGNet 和 TavernNet 独立子网
- 遵循蛛网式分布式设计
- 通过 M-Link 通信

✅ **记忆隔离**
- 两个模式记忆完全独立
- 不互相干扰
- 各自持久化

✅ **核心集成**
- 情绪系统集成
- 记忆系统集成
- 人格系统集成

### 工具注册

✅ **统一管理**
- 所有工具注册到 ToolNet
- 统一的调用接口
- 规范的参数验证

✅ **工具总数**
- TRPG 工具：11 个
- 酒馆工具：7 个
- 总计：49 个（含其他子网工具）
