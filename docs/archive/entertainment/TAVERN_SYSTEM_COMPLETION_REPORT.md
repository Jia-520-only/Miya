### 5. 情绪与氛围系统

✅ **5 种情绪氛围**
1. **warm** (温暖)
   - 描述：亲切、关怀
   - 场景：壁炉火苗、烤面包香气

2. **relaxed** (轻松)
   - 描述：幽默、俏皮
   - 场景：爵士乐流淌、轻松惬意

3. **serious** (严肃)
   - 描述：认真、专注
   - 场景：安静严肃、适合思考

4. **dark** (暗黑)
   - 描述：神秘、阴郁
   - 场景：烛光摇曳、昏暗神秘

5. **healing** (治愈)
   - 描述：温柔、安抚
   - 场景：暖色调灯光、抚平伤痕
