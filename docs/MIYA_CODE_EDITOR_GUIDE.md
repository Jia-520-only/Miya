# 弥娅代码编辑器使用指南

## 功能概述

弥娅代码编辑器是一个功能齐全的代码编辑界面，提供了与 VS Code 类似的编辑体验，并深度集成了弥娅 AI 系统。

## 主要功能

### 1. 文件资源管理器
- **文件树浏览**: 展示项目文件结构，支持展开/折叠目录
- **文件筛选**: 通过输入框快速筛选文件
- **显示隐藏文件**: 可选择是否显示隐藏文件（以 . 开头的文件）
- **文件图标**: 根据文件类型显示不同图标

### 2. Monaco 编辑器
- **语法高亮**: 支持多种编程语言
- **智能提示**: 自动补全和参数提示
- **代码格式化**: 一键格式化代码
- **自动保存**: 实时检测未保存的更改
- **多光标编辑**: 支持多光标操作
- **小地图**: 右侧显示代码缩略图

### 3. 支持的语言
- TypeScript / JavaScript
- Python
- Java
- C / C++ / C#
- Go / Rust
- PHP / Ruby
- HTML / CSS / SCSS
- JSON / XML / YAML
- Markdown
- SQL
- Shell / Batch / PowerShell

### 4. 终端集成
- **终端面板**: 底部集成终端窗口
- **输出面板**: 查看代码执行输出
- **实时输出**: 运行代码时实时显示结果
- **历史记录**: 保留终端执行历史

### 5. 快捷功能
- **格式化代码**: 点击格式化按钮或使用快捷键
- **新建文件**: 创建新的代码文件
- **保存文件**: 保存当前编辑的文件
- **运行代码**: 快速运行 JavaScript/Python 代码
- **切换终端**: 显示/隐藏底部终端面板

## 使用方法

### 打开代码编辑器
1. 启动弥娅桌面应用
2. 点击顶部导航栏的"代码"标签

### 编辑文件
1. 在左侧文件树中点击文件名打开
2. 在编辑器中修改代码
3. 点击"保存"按钮（或 Ctrl+S）保存更改
4. 未保存的文件会显示●标记

### 新建文件
1. 点击工具栏的"新建文件"按钮
2. 输入文件名（包括扩展名）
3. 系统会自动根据扩展名设置语言

### 运行代码
1. 编写或打开代码文件
2. 点击工具栏的"运行"按钮
3. 在底部终端查看执行结果

### 使用终端
1. 点击工具栏的"终端"按钮
2. 在终端面板中执行命令
3. 可以切换到"输出"标签查看程序输出

## 高级功能

### 代码片段
Monaco 编辑器内置了多种语言的代码片段，输入前缀后按 Tab 键可展开。

### 多光标编辑
- 按 Alt+点击可添加多个光标
- 按 Ctrl+Alt+向上/向下箭头可向上/向下添加光标

### 主题切换
编辑器支持深色主题，与弥娅的整体界面风格保持一致。

### 自动完成
编辑器会根据上下文提供智能代码补全建议。

## 技术细节

### 核心技术
- **Monaco Editor**: VS Code 的核心编辑器组件
- **Vue 3**: 响应式前端框架
- **TypeScript**: 类型安全的 JavaScript 超集
- **Electron**: 跨平台桌面应用框架

### 后端集成
编辑器通过弥娅的后端 API 实现以下功能：
- 文件读取和保存
- 代码执行
- 终端命令执行
- 文件系统浏览

### 扩展性
编辑器架构设计为模块化，可以轻松添加：
- 新的语言支持
- 自定义主题
- 额外的编辑器功能
- 集成其他工具

## 与弥娅 AI 的集成

虽然代码编辑器本身不直接调用 AI，但可以：
1. 在"对话"页面让 AI 生成代码
2. 切换到"代码"页面查看和编辑代码
3. 在编辑器中运行和测试代码
4. 回到"对话"页面询问问题和获取帮助

## 常见问题

### Q: 为什么运行代码没有输出？
A: 确保点击了"终端"按钮打开终端面板，输出会显示在终端或输出标签中。

### Q: 如何更改文件的语言？
A: 点击工具栏中的语言选择下拉框，选择正确的语言。

### Q: 编辑器支持哪些快捷键？
A: 大部分 VS Code 的快捷键都支持，具体请参考上表。

### Q: 如何撤销未保存的更改？
A: 按 Ctrl+Z 可以撤销更改，或重新打开文件恢复到上次保存的状态。
