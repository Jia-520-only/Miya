## 🎯 弥娅架构理念

### 核心设计原则

1. **五层认知架构**
   - 第一层：弥娅内核 (core/) - 人格、伦理、身份
   - 第二层：蛛网主中枢 (hub/) - 记忆、情绪、决策、调度
   - 第三层：M-Link统一传输链路 (mlink/) - 五流分发、消息路由
   - 第三层：弹性分支子网集群 (webnet/) - 多领域子网
   - 第四层：感知环 + 注意力闸门 (perceive/) - 全域感知
   - 第五层：演化沙盒 (evolve/) - 实验和共演

2. **蛛网式分布式设计**
   - M-Link五流分发（指令流/感知流/同步流/信任流/记忆流）
   - 弹性分支子网热插拔
   - 动态路径评分和路由

3. **记忆-情绪耦合**
   - 记忆-情绪双向影响
   - 情绪染色机制
   - 潮汐记忆+梦境压缩

4. **人格恒定机制**
   - 五维人格向量
   - 熵监控防异化
   - 伦理约束和底线
